For Senior Design, our team developed a wireless charging system for 13.6V, 6000mAh E-Bikes. The system has the ability to easily scale to many bikes in a row and when compared to wired charging our system is around 70% efficient. This solution can be used in cities where mass bike racks can be built and the bike racks offer an easy interface for users.
Optimized Forward-Pass Convolution Layers for LeNet-5 Neural Networks using CUDA November 2020 - Decemeber 2020
For ECE 408, our team optimized forward-pass convolution layers for LeNet-5 neural networks using CUDA. We used the Mini-DNN framework for implementing the modified LeNet-5. We implemented the following optimizations. Unrolling and shared memory matrix multiply, shared memory convolution, kernel fusion, weight matrix in constant memory, restricts and loop unrolling, optimized tile widths, Exploiting parallelism on input channels, Multiple kernel implementations for different layer sizes, input channel reduction using a tree and FP16 arithmetic.
